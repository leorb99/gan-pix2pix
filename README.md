# GAN (Generative Adversarial Network) - Pix to Pix

Este trabalho consiste em usar o algoritmo descrito no artigo 'Image-based Plant Disease Diagnosis with Unsupervised Anomaly
Detection Based on Reconstructability of Colors'. O algoritmo deve treinar uma GAN para detectar anomalias em folhas de plantas. Após o treinamento, um dataset com imagens de folhas deve ser lido e a rede neural deve reproduzir a imagem e aplicar um algoritmo que compara as imagens e classifica o quão diferente as imagens são. Após esse processo ele classifica se a folha está ou não saudável.
